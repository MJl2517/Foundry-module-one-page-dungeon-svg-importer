# Watabou One Page Dungeon Importer

Модуль для Foundry VTT v13, который импортирует SVG-карты из генератора [Watabou One Page Dungeon](https://watabou.itch.io/one-page-dungeon) и автоматически создает сцену со стенами, обычными дверями и секретными дверями.

Модуль рассчитан на быстрый перенос карты из Watabou в Foundry без отдельного JSON-импорта.

## Возможности

- Добавляет кнопку импорта во вкладку сцен Foundry.
- Работает только для GM.
- Импортирует один `.svg` файл Watabou через drag and drop или выбор файла в зоне загрузки.
- Создает новую сцену Foundry.
- Использует SVG как фон сцены.
- Обрезает фон сцены до области карты с небольшим отступом.
- Автоматически находит размер сетки Watabou внутри SVG и совмещает ее с сеткой Foundry.
- Размер клетки Foundry по умолчанию: `72`.
- Строит стены по маске пола Watabou.
- Распознает обычные двери Watabou и создает закрытые Foundry doors.
- Распознает часть секретных проходов Watabou и создает Foundry secret doors.
- Для больших карт автоматически уменьшает итоговый размер сцены, чтобы фон не превращался в черный прямоугольник из-за лимитов текстур.
- Поддерживает русскую и английскую локализацию.

## Как пользоваться

1. В Watabou One Page Dungeon сгенерируйте карту.
2. Перед экспортом отключите `Notes`, если они включены.
3. Экспортируйте карту в SVG.
4. В Foundry откройте вкладку сцен.
5. Нажмите кнопку импорта Watabou.
6. Перетащите SVG в окно импорта.
7. При необходимости измените название сцены и размер клетки.
8. Нажмите `Импорт`.

После импорта появится новая сцена с фоном, стенами и дверями.

## Размер клетки

Поле `Размер клетки` задает размер одной клетки Foundry в пикселях. Значение по умолчанию: `72`.

Модуль сам определяет размер сетки внутри SVG Watabou и масштабирует карту так, чтобы клетки Watabou совпадали с клетками Foundry. Обычно достаточно оставить `72`.

Если карта очень большая, модуль может автоматически уменьшить итоговый размер клетки сцены. Это нужно, чтобы SVG-фон не превышал безопасный размер текстуры Foundry и не отображался черным.

## Что импортируется

### Фон сцены

Модуль сохраняет исходный SVG как фон сцены, но создает для него cropped viewBox вокруг самой карты. Лишние элементы листа и большие пустые поля не должны попадать в сцену.

### Стены

Стены строятся по геометрии пола Watabou. Модуль ищет слой маски пола в SVG и на его основе создает Foundry walls.

### Обычные двери

Обычные двери определяются по SVG-формам дверей Watabou. Такие двери создаются как закрытые Foundry doors.

### Секретные двери

Watabou не всегда хранит тип двери в SVG так, чтобы его можно было надежно восстановить. Поэтому модуль не читает отдельный тип `locked`, `hidden` или `secret` из данных Watabou.

Вместо этого модуль пытается распознавать секретные проходы по геометрии: обычно это узкие проходы примерно в половину клетки, которые ведут через толщу стены в скрытую комнату. В таких местах модуль строит короткий коридор и ставит Foundry secret door на линии входа.

Это эвристика. Она работает для типичных секретных проходов Watabou, но нестандартные стили или необычная геометрия могут потребовать ручной правки стен после импорта.

## Ограничения и нюансы

### Поддерживается только SVG

Модуль не использует JSON-экспорт Watabou. Для импорта нужен только `.svg` файл.

### Карты под наклоном не поддерживаются

Не импортируйте карты, где подземелье повернуто или находится под наклоном. Квадратную сетку Foundry нельзя корректно совместить с произвольно повернутой картой Watabou.

Если модуль обнаружит такой SVG, импорт будет остановлен с ошибкой.

### Отключайте Notes в Watabou

Перед экспортом SVG рекомендуется отключать `Notes`. Они могут добавлять подписи и элементы листа, которые не нужны для сцены Foundry.

### Визуальные артефакты SVG Watabou

Иногда Watabou экспортирует SVG с визуальными артефактами: случайными черными линиями, полосами или другими дефектами изображения. Если такие линии уже есть внутри SVG или появляются из-за особенностей SVG-экспорта Watabou, это не ошибка стен модуля.

Модуль строит стены Foundry отдельно от изображения. Поэтому визуальный дефект на фоне может быть виден, даже если стены и двери импортированы правильно.

### Разные стили Watabou

Watabou может менять SVG-структуру в зависимости от выбранного стиля карты. Модуль поддерживает несколько вариантов дверей, включая геометрическое распознавание прямоугольных дверей, но некоторые стили могут потребовать доработки распознавания.

## Установка

### Через Foundry

После публикации первого релиза модуль можно будет установить через manifest URL:

```text
https://github.com/MJl2517/Foundry-module-one-page-dungeon-svg-importer/releases/latest/download/module.json
```

### Вручную

Скопируйте папку `watabou-one-page-dungeon-importer` в папку модулей Foundry:

```text
Foundry Data/Data/modules/
```

После этого включите модуль в настройках мира Foundry.

## Разработка

Проект не требует build step и внешних зависимостей. Основной код находится в папке:

```text
watabou-one-page-dungeon-importer/
```

В репозитории может быть локальная папка `secret/` с тестовыми картами, скриптами или личными файлами. Она не должна попадать в GitHub.

Также не нужно коммитить собранные архивы и временные файлы релизов.

## Совместимость

- Foundry VTT: v13
- Импорт: Watabou One Page Dungeon SVG export

Модуль находится в активной разработке. Основной импорт сцен, стен, обычных дверей и типичных секретных проходов уже работает, но SVG-структура Watabou может отличаться между стилями и версиями генератора.
